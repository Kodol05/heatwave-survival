# 폭염 게임 기획서

## 1. 개요

| 항목 | 내용 |
|---|---|
| 플랫폼 | 웹 브라우저 |
| 엔진 | Phaser 3 |
| 언어 | JavaScript |
| 타겟 | 어린이/학생 |
| 플레이 시간 | 1~3분 (짧은 즉석 플레이) |
| 목적 | 폭염 대처법 학습 + 재미 |

---

## 2. 전체 구조

- **허브**: 집 (메인 화면)
- **미니게임**: 총 3개 슬롯
  - 어머니 NPC → 심부름 게임 (확정)
  - 컴퓨터 → 응급처치 게임 (확정)
  - TV → 보류 (3번째 미니게임 진입점 후보)

---

## 3. 미니게임 ①: 심부름 게임 (메인)

### 3.1 진입 흐름

1. 집 화면에서 **어머니 NPC**에게 말 걸기
2. 어머니가 목적지를 알려줌 (목적지는 게임상 중요하지 않음, 플레이버 텍스트)
3. **시간 선택 화면** 표시
4. 8~20시 중 출발 시간 선택
5. 각 시간대별로 **난이도 + 설명** 표시
   - 예: "14시 - 36°C - 어려움 / 폭염 절정 시간! 그늘이 적고 더위가 빨리 올라가요"
6. 선택 시 심부름 게임 시작

### 3.2 시간대별 온도 테이블

| 시간 | 온도 | 난이도 |
|---|---|---|
| 8~9시 | 26°C | 매우 쉬움 |
| 10~11시 | 30°C | 쉬움 |
| 12~13시 | 34°C | 보통 |
| 14~15시 | 36°C | 어려움 (폭염 절정) |
| 16~17시 | 33°C | 보통 |
| 18~19시 | 29°C | 쉬움 |
| 20시 | 26°C | 매우 쉬움 |

### 3.3 게임 메커니즘

- **장르**: 쿠키런 스타일 자동 전진 러너
- **조작**: 상하 자유 이동 (중력 없음)
- **클리어 조건**: 결승점 도착
- **게임 오버 조건**: 더위 100 또는 수분 0 도달

### 3.4 게이지 시스템

- **시작값**: 더위 낮음 + 수분 높음
  - 권장 초기값: 더위 0~20, 수분 80~100
- **수분**: 시간이 지나면서 자동 감소
- **연동 규칙**: 더위가 높을수록 수분 감소 속도가 빨라짐
- 두 게이지 중 하나라도 위험 수치 도달 시 게임 오버

### 3.5 지형 시스템 (3종류)

| 지형 | 효과 | 빈도 |
|---|---|---|
| 그늘 | 더위 회복 | 자주 등장 |
| 일반 길 | 더위 약하게 ↑ (양패) | 가장 자주 등장 |
| 아스팔트 | 더위 강하게 ↑ (소패) | 가끔 등장 |

### 3.6 난이도 조절 (온도 연동)

- 시간대 온도가 높을수록 → **일반 도로/아스팔트에서 더위 상승 속도가 빨라짐**
- 그늘 회복 속도는 변하지 않음 → 한낮엔 그늘 의존도가 훨씬 높아짐
- "한낮은 위험해" 메시지가 자연스럽게 체감되는 구조

### 3.7 아이템 (7종)

획득 시 자동 적용 (인벤토리 없음).

| 아이템 | 수분 | 더위 | 학습 메시지 |
|---|---|---|---|
| 💧 물 | ↑↑ | ↓ | 가장 좋은 선택 |
| 🧊 얼음물 | ↑ | ↓↓ | 더위 잡기 최고 |
| 🥤 이온음료 | ↑↑↑ | ↓ | 땀 많이 흘렸을 때 최고 |
| 🥒 수박 | ↑↑ | ↓ | 제철 과일이 도움 |
| ☕ 핫초코 | ↑ | ↑ | 여름엔 따뜻한 거 피하기 (함정) |
| 🍜 라면 | ↑ | ↑↑ | 더위에 뜨거운 음식 위험 (함정) |
| 🪭 부채 | - | ↓ | 즉시 시원하게 |

### 3.8 점수 계산

```
점수 = 남은 수분 게이지 + 남은 더위 여유 + 클리어 보너스
```

### 3.9 클리어 후 흐름

- 집으로 자동 복귀
- 점수 + 획득 아이템 결과 화면 표시
- 다시 시작 또는 다른 미니게임 선택 가능

## 4. 미니게임 ②: 응급처치 게임

### 4.1 진입 흐름

1. 집 화면에서 **컴퓨터**에 상호작용
2. 응급처치 게임 시작

### 4.2 게임 메커니즘

- **장르**: 순발력 + 단답 선택 게임
- **한 판 구성**: 5개 상황 연속 출제
- **상황당 선택지**: 4개 (정답 1 + 오답 3)
- **타이머 바**: 모두 동일한 길이로 줄어듦

### 4.3 판정 시스템

| 판정 | 조건 | 점수 |
|---|---|---|
| Perfect | 빠르게 정답 선택 | 높은 점수 |
| Good | 늦게 정답 선택 | 중간 점수 |
| Miss | 시간 초과 | 0점 |
| 오답 | 틀린 선택지 클릭 | 감점/Miss |

### 4.4 5개 상황 (난이도 순)

1. **친구가 운동장에서 어지럼증 호소**
   - 정답: 그늘로 옮기고 119 신고
2. **친구가 의식을 잃고 쓰러짐**
   - 정답: 즉시 119 신고
3. **땀이 안 나고 피부 뜨겁고 건조함 (열사병 증상)**
   - 정답: 시원한 곳으로 옮기고 몸을 식힘
4. **운동 후 다리에 쥐가 남 (열경련)**
   - 정답: 시원한 곳 휴식 + 수분 섭취
5. **에어컨 없는 방에서 노인이 더위 호소**
   - 정답: 119 신고 + 시원한 곳으로

### 4.5 점수 계산

```
총점 = 5개 상황 각각의 판정 점수 합계
```

## 5. 학습 메시지 요약

게임 전반에서 자연스럽게 전달되는 폭염 대처법:

1. 한낮(14~15시) 외출은 매우 위험
2. 그늘이 큰 도움이 됨
3. 아스팔트 같은 뜨거운 지형은 피하기
4. 더위가 심할수록 수분 보충 더 자주
5. 여름엔 시원한 음료/음식이 좋고, 뜨거운 음식은 피하기
6. 온열질환 단계별 증상과 대처법 (어지럼증 → 열탈진 → 열사병)
7. 응급 상황엔 119 신고
